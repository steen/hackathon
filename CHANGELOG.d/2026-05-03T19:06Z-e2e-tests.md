### Added

- **tests/e2e**: Phase-2 end-to-end suite that boots the real `apps/server`
  binary on a random port with a tempdir SQLite + ephemeral
  `CHAT_INVITE_CODE` / `CHAT_JWT_SECRET`, then drives the real `chatd` CLI
  and the real web app (Playwright + chromium against the Vite dev server)
  through register → login → channel create → post → echo, watch + reconnect
  on server restart, logout invalidation, presence join/leave, two-context
  cross-receive, WS drop+restore, and CLI ↔ web cross-client interop.
  Run via `pnpm e2e` (CLI suite + browser suite), or `pnpm e2e:cli` /
  `pnpm e2e:web` individually. CI invokes `pnpm e2e` in a new `e2e` job
  that installs Playwright browsers via
  `pnpm --filter web exec playwright install --with-deps chromium`.
  Closes #81. (2026-05-03T19:06Z)
