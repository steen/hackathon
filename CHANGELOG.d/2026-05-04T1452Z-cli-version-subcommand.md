- chatd CLI: add `chatd version`, `chatd --version`, and `chatd -v`. Output is a single line with the module version (or `dev` for untagged builds), short VCS revision (with a `dirty` marker when the working tree was modified), Go runtime version, and `GOOS/GOARCH`. Build metadata is read via `runtime/debug.ReadBuildInfo`, so binaries produced by `go install` or a tagged release surface the tag without needing `-ldflags`. (#151)
